# Aurelia Kinetic Design System

A creative recombination of the three supplied design-system projects:

- **Open Design**: warm editorial paper, collage imagery, side rails, coordinate-like metadata, `data-reveal`, headroom navigation and wire marquee language.
- **Future UI**: industrial display typography, orbital geometry, floating shapes, gradient borders and the original `float1`, `float2`, `float3`, `spin1`, `spin2` animation names.
- **Glass Pricing**: `.ds-nav`, `.ds-section`, `.ds-card`, `.ds-spec-row`, `.glass-effect`, `.glass-button`, `.gradient-text`, component documentation and active navigation behavior.

## Run

Open `index.html` directly in a modern browser. The page has no build step. Google Fonts are requested from CDN and gracefully fall back to system fonts.

## Files

- `index.html`: complete interactive design-system showcase
- `assets/css/styles.css`: tokens, layouts, components, responsive rules and animation keyframes
- `assets/js/app.js`: reveal observer, headroom nav, active sections, modal focus management, magnetic controls, tilt, ripple, copy helpers and motion controls
- `assets/images/`: optimized visual assets recombined from the supplied archives

## Accessibility

The system supports keyboard focus, modal focus trapping, Escape-to-close, `prefers-reduced-motion`, a manual pause control and contrast targets documented inside the color section.
